# WynGen

Chrome-расширение для генерации паролей с заданной длиной прямо в браузере.

## Установка

1. Скачай [`wyngen-chrome.crx`](wyngen-chrome.crx)
2. Открой `chrome://extensions/`
3. Включи режим разработчика
4. Перетащи `.crx` в окно расширений

## Возможности

- Задать минимальную и максимальную длину пароля
- Генерация из букв (A–Z, a–z), цифр и спецсимволов
- Копирование пароля кликом

## Стек

- Vanilla JS
- Chrome Extensions Manifest V2
